责任链模式

发表于 2017-07-16 12:10:18 | 分类于 设计模式 | 阅读次数
字数统计 600 | 阅读时长 3

责任链：每个对象有对其下家的引用而形成的一条链
当前对象自己条件不符合或处理不了，
交由下家对象处理，一直传递下去直到处理完请求
下家对象是什么鬼：就是new出自家的时候化设置的下家对象
